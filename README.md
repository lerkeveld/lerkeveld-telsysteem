# lerkeveld-telsysteem

Setup:
add the fontawesome fa-solid-900.ttf file to src/main/resources

Run the program in linux with:
```
./gradlew run
```
and in windows:
```
gradlew.bat run
```

Build the application jar in java with
```
./gradlew jar
```
and in windows:
```
gradlew.bat jar
```

The build jar is stored in build/libs
